# CG_MINI_PROJECT
A Computer graphics mini project named Rubik's Cube.
## COMMANDS
The commands to interact with the Rubik's Cube are:

L: rotate right

J: rotate left

I: rotate down

K: rotate up

+: zoom in

-: zoom out

mouse left click: zoom in

mouse right click: zoom out

Moving/Rotating the layers

Along x axis

Q: select 1st layer

W: select 2nd layer

E: select 3rd layer

Along y axis

A: select 1st layer

S: select 2nd layer

D: select 3rd layer

Along z axis

Z: select 1st layer

X: select 2nd layer

C: select 3rd layer

Face Rotation

O: rotate selected face clockwise

U: rotate selected face counter-clockwise
